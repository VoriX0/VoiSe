# VoiSe Gate 5.2 — SoundBoard Transport & Timeline

Gate 5.2 доводит транспорт SoundBoard до более продуктового поведения и внешнего вида.

## Что изменено после Gate 5.1

- заголовок окна: `VoiSe Gate 5.2 - SoundBoard Transport & Timeline`;
- таймлайн перенесён на отдельную строку ниже статуса трека;
- таймлайн теперь поддерживает перемотку текущего звука;
- кнопки `Play` и `Pause` объединены в одну кнопку с иконкой:
  - `▶` — запустить или продолжить;
  - `⏸` — поставить на паузу;
- кнопки предыдущего и следующего трека заменены на иконки `⏮` и `⏭`;
- кнопка `Stop` оставлена текстовой и полностью останавливает звук;
- кнопки `Previous`/`Next` расположены сверху в паре, `Stop` — под ними на ширину двух кнопок;
- кнопка `Play/Pause` расположена слева от таймлайна;
- увеличен отступ между выпадающим списком категории и списком треков;
- bottom-информация SoundBoard перенесена в нижнюю область, где раньше был общий лог;
- лог приложения перенесён во вкладку `Settings`.

## Запуск

```powershell
dotnet run --project src/VoiSe.App
```

## Проверка

1. В заголовке окна должно быть `VoiSe Gate 5.2`.
2. Добавь трек в SoundBoard или выбери существующий.
3. Нажми `▶` — звук должен начать воспроизводиться.
4. Во время воспроизведения кнопка должна показывать `⏸`.
5. Нажми `⏸` — звук должен встать на паузу.
6. Нажми `▶` ещё раз — звук должен продолжиться с того же места.
7. Перемести таймлайн — звук должен перейти к выбранному месту.
8. Нажми `Stop` — звук должен полностью остановиться.
9. Нажми `⏮` / `⏭` — должен выбраться и запуститься предыдущий / следующий трек в категории.
10. Проверь, что bottom-панель показывает количество категорий, звуков и счётчик использования.
