# Gate 7.0 — Scenes Foundation

Initial implementation of Scenes.

## Added

- Functional Scenes tab.
- Capture current VoiSe state as a scene.
- Apply selected scene.
- Update selected scene from current state.
- Rename and delete scenes.
- Open scenes folder.
- One JSON file per scene in `%LOCALAPPDATA%\VoiSe\scenes\`.

## Captured state

- Voice Changer sliders.
- Last applied voice preset name when available.
- Voice Monitor state.
- SoundBoard selected category.
- SoundBoard selected sound.
- Virtual Mic Master.
- SoundBoard → Virtual Mic volume.
- SoundBoard → Headphones volume.
- SoundBoard virtual mic delay.

## Not yet implemented

- Autoplaying loop layers.
- Multi-sound one-shot scene buttons.
- Scene hotkeys.
- Scene import/export tools.
