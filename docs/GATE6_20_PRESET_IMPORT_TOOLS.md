# Gate 6.20 — Preset Import Tools

Final Gate 6 polish before Scenes.

- Added a preset tools tile after `New preset`.
- The tools tile is split vertically into two equal buttons: `Import` and `Folder`.
- `Import` lets the user pick a voice preset JSON and copies it into the VoiSe presets directory.
- `Folder` opens `%LOCALAPPDATA%\VoiSe\presets\` in Explorer.
- Existing preset hotkeys, JSON export, and SoundBoard scroll behavior are unchanged.
