# Gate 6.19 — Hotkey Button Capture UX

- Hotkey assignment uses a button instead of a read-only text box.
- Click the button, press a key or Ctrl/Alt/Shift combination, or press Esc to cancel.
- The adjacent × button clears the hotkey.
- Only plain A-Z and < > { } hotkeys are local-only; other explicit single keys and modifier combinations can be global.
- Play/Pause remains a single transport hotkey.
