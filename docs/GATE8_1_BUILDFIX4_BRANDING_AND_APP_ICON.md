# Gate 8.1 buildfix 4 — branding polish and app icon

This buildfix adds final release-style branding polish over Gate 8.1 buildfix 3.

## Included changes

- Added the provided VoiSe logo as the application icon.
- Removed the white outer corners by converting the external white background to transparency.
- Added app icon assets in multiple sizes and an `.ico` file for the executable/window icon.
- Updated the custom title bar to display the logo icon.
- Updated the main title and window title from `Gate` wording to `Version` wording.
- Removed the descriptive subtitle under the main title.

## Display version

The UI now shows `VoiSe Version 8.1.4`.
