# Gate 6.12 — Settings Log Fullscreen

## Changes

- Reduced the Settings log panel height again.
- Kept the log wheel zone compact so it does not overlap the Settings controls above it.
- Added an **Open fullscreen** button for the application log.
- Reflowed Voice Changer sliders to 4 columns per row.
- Preserved the known-good SoundBoard scroll behavior from Gate 6.8 / 6.5.

## Test checklist

1. Build and run the app.
2. Confirm the window title shows Gate 6.12.
3. Confirm Voice Changer sliders are arranged 4 per row.
4. Confirm Voice Changer still scrolls down to the fullscreen bottom area.
5. Confirm Settings log is shorter and does not steal scroll from Settings controls above.
6. Click **Open fullscreen** in Settings log and confirm the log opens in a large dialog.
7. Confirm SoundBoard scrolling is unchanged from Gate 6.8 / 6.5.
